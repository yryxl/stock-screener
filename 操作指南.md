# 芒格价值投资选股系统 - 操作指南

> 本指南面向完全没有开发经验的用户，每一步都有详细说明。

---

## 你需要做的事（共5步，约20分钟）

| 步骤 | 内容 | 耗时 |
|------|------|------|
| 第1步 | 获取微信推送token | 1分钟 |
| 第2步 | 注册GitHub账号 | 3分钟 |
| 第3步 | 创建仓库并上传代码 | 5分钟 |
| 第4步 | 配置自动运行 | 3分钟 |
| 第5步 | 部署持仓管理界面 | 5分钟 |

---

## 第1步：获取微信推送 Token

这一步让系统能给你的微信发消息。

1. 打开微信，搜索公众号 **「PushPlus推送加」**
2. 关注这个公众号
3. 关注后，公众号会自动回复你一个 **token**（一串字母数字）
4. 也可以点公众号菜单里的「个人中心」查看你的 token
5. **把这个 token 记下来**，后面要用

---

## 第2步：注册 GitHub 账号

GitHub 是一个代码托管平台，我们用它来：
- 存放代码（免费）
- 每天自动运行选股分析（免费）

1. 打开浏览器，访问 https://github.com
2. 点击右上角 **Sign up**
3. 填写邮箱、密码、用户名，完成注册
4. 验证邮箱（GitHub会发一封验证邮件到你的邮箱）

### 创建 Personal Access Token（让网页界面能读写数据）

1. 登录 GitHub 后，点击右上角你的头像 → **Settings**
2. 左侧菜单拉到最底部，点 **Developer settings**
3. 点 **Personal access tokens** → **Tokens (classic)**
4. 点 **Generate new token** → **Generate new token (classic)**
5. 设置：
   - **Note**：填 `stock-screener`（备注用途）
   - **Expiration**：选 **No expiration**（永不过期）
   - **勾选权限**：只需要勾选 **repo**（第一个选项，包含所有子选项）
6. 点底部 **Generate token**
7. **立刻复制这个 token！**（以 `ghp_` 开头的一串字符）
   - ⚠️ 这个 token 只显示一次，离开页面就看不到了
   - 把它和 PushPlus token 一起保存好

---

## 第3步：创建仓库并上传代码

### 方式一：通过网页上传（最简单）

1. 登录 GitHub，点左上角 **+** → **New repository**
2. 设置：
   - **Repository name**：填 `stock-screener`
   - **选 Private**（私有仓库，别人看不到你的持仓）
   - 其他不用动
3. 点 **Create repository**
4. 在新建的空仓库页面，点 **uploading an existing file**
5. 把以下文件**全部拖进去**（从 `stock_screener` 文件夹里）：
   ```
   config.yaml
   requirements.txt
   data_fetcher.py
   screener.py
   notifier.py
   main.py
   app.py
   holdings.json
   .gitignore
   ```
6. 在页面底部 Commit changes 填写 `初始化`，点 **Commit changes**

### 上传 GitHub Actions 工作流文件

7. 在仓库页面点 **Add file** → **Create new file**
8. 文件名输入：`.github/workflows/daily_screen.yml`
   （输入 `.github/` 后会自动创建文件夹）
9. 把 `.github/workflows/daily_screen.yml` 的内容复制粘贴进去
10. 点 **Commit changes**

### 上传 Streamlit 配置文件

11. 同样方式创建 `.streamlit/config.toml`，粘贴内容并提交

## 第4步：配置自动运行

### 设置 PushPlus Token（让 GitHub Actions 能推送微信）

1. 在你的仓库页面，点顶部 **Settings**
2. 左侧菜单点 **Secrets and variables** → **Actions**
3. 点 **New repository secret**
4. 设置：
   - **Name**：填 `PUSHPLUS_TOKEN`
   - **Secret**：粘贴你在第1步获取的 PushPlus token
5. 点 **Add secret**

### 测试运行

1. 在仓库页面点顶部 **Actions**
2. 左侧会看到 **「每日选股分析」**
3. 点进去，右侧点 **Run workflow** → **Run workflow**
4. 等几分钟，看运行结果
   - 绿色 ✅ = 成功
   - 红色 ❌ = 失败（点进去看日志排查原因）
5. 如果成功，检查你的微信是否收到了 PushPlus 的消息

> 注意：首次运行会扫描全A股，可能需要较长时间。之后每个工作日下午5点自动运行。

---

## 第5步：部署持仓管理界面

这一步让你有一个网页来管理持仓（增删改查）。

1. 打开 https://share.streamlit.io （用 GitHub 账号登录）
2. 点 **New app**
3. 设置：
   - **Repository**：选你的 `stock-screener`
   - **Branch**：`main`
   - **Main file path**：`app.py`
4. 点 **Advanced settings**，在 **Secrets** 区域粘贴以下内容：
   ```toml
   [github]
   token = "ghp_你的GitHub_Token"
   repo = "你的GitHub用户名/stock-screener"
   holdings_file = "holdings.json"
   ```
   把 `ghp_你的GitHub_Token` 换成第2步获取的 GitHub Token
   把 `你的GitHub用户名` 换成你的 GitHub 用户名
5. 点 **Deploy!**
6. 等1-2分钟，部署完成后你会得到一个网址，类似：
   `https://你的用户名-stock-screener.streamlit.app`
7. **把这个网址收藏到手机浏览器书签里**，以后随时打开管理持仓

## 日常使用

### 添加/修改持仓
打开 Streamlit 网址 → 添加/修改/删除股票

### 收到微信通知
- 系统每个工作日自动分析
- 有买入或卖出信号时，微信会收到 PushPlus 的推送消息
- **没有信号时不会打扰你**

### 手动触发分析
GitHub 仓库 → Actions → 每日选股分析 → Run workflow

## 常见问题

### Q：微信没收到消息？
1. 确认你关注了 PushPlus 公众号
2. 在 GitHub 仓库 Settings → Secrets 检查 PUSHPLUS_TOKEN 是否正确
3. 在 Actions 里查看运行日志是否有报错

### Q：持仓界面打不开？
1. 检查 Streamlit Cloud 是否正常部署
2. 检查 Secrets 中的 GitHub Token 和 repo 是否正确

### Q：想修改筛选条件？
编辑仓库里的 `config.yaml` 文件，修改对应的数值即可。
比如把股息率门槛从 6% 改为 5%：
```yaml
dividend_yield_min: 5
```

### Q：GitHub Actions 没有自动运行？
1. GitHub 对免费账户的 Actions 有时会延迟
2. 确保仓库 Settings → Actions → General 中，Workflow permissions 设为 **Read and write permissions**

## 文件说明

| 文件 | 作用 |
|------|------|
| `config.yaml` | 配置文件（筛选条件、卖出规则） |
| `data_fetcher.py` | 从 AKShare 获取 A 股数据 |
| `screener.py` | 五大指标筛选引擎 |
| `notifier.py` | 微信推送（PushPlus） |
| `main.py` | 主程序（每天自动运行） |
| `app.py` | Streamlit 持仓管理界面 |
| `holdings.json` | 你的持仓数据 |
| `.github/workflows/daily_screen.yml` | GitHub Actions 自动运行配置 |
