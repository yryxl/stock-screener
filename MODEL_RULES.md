# 选股模型完整规则清单

本文档记录当前模型的所有规则、阈值、决策逻辑，用于日后回溯、维护、调整。
最后更新：2026-04-10 | commit 08ffd6a

## 核心理念（不可违反）

1. **宁可错过，不可犯错**（芒格）—— 所有规则设计偏保守
2. **净资产收益率是核心**，其他指标是辅助（巴菲特 1979 年致股东信）
3. **现金流不会骗人**（巴菲特 1986 年致股东信"股东盈余"概念）
4. **长期持有，不频繁交易**（买后除非明确卖出信号，否则不动）
5. **不投大宗商品周期股**（巴菲特/芒格"时间是优秀企业的朋友"）

## 二、数据采集（backtest_collector.py）

### 采集来源
- akshare 库：月度价格（新浪）、PE(TTM)（百度）、财务指标（同花顺）、分红（东财）
- 按列名精确匹配同花顺的财务报表（兼容银行股列序不同）

### 采集字段（financial_data）
每个年报/季报 record 包含：
- `date`：报告期（"YYYY-MM"）
- `roe`：净资产收益率
- `gross_margin`：销售毛利率（银行股为 None）
- `net_margin`：销售净利率
- `debt_ratio`：资产负债率
- `current_ratio`：流动比率（银行股为 None）
- `eps`：基本每股收益
- `revenue_growth`：营业总收入同比增长率
- `profit_growth`：净利润同比增长率
- `ocf_per_share`：**每股经营现金流**（用于现金流质量检查）

### 月度快照生成
- 披露滞后：X 年 12 月的年报要到 X+1 年 4 月才可见
- **只取年报**（12 月数据），不用季报（避免季度 ROE 被当年化 ROE 误判）
- **数据范围**：2001 年 1 月 ~ 2025 年 12 月，共 300 个月度快照
- 老股票（茅台、五粮液等）可回溯到 2001 年上市时
- 新股票（宁德时代 2018、中芯 2020 等）自上市后开始有数据
- 退市股（乐视、康美等）仅有退市前的数据，退市后月份为空

## 三、行业分类（screener.py INDUSTRY_PE）

每个行业配置：`{low, fair_low, fair_high, high, type, complexity, high_leverage?}`

- `type`: simple / consumer / mature / utility / tech / growth / cycle
- `complexity`: simple / medium / complex（影响 ROE 门槛）
- `high_leverage`: True（银行/保险/券商/地产）—— 豁免负债率检查和杠杆惩罚

### 关键分类
- **白酒 / 家电 / 中药**：simple + consumer
- **银行 / 保险**：mature + simple + high_leverage，PE 区间 5-12
- **证券**：mature + medium + high_leverage
- **地产**：mature + medium + high_leverage（不是 cycle！地产 ROE 下滑是真恶化）
- **锂电 / 新能源 / 电池**：**cycle** + complex（大宗商品周期）
- **石油 / 煤炭 / 钢铁 / 稀土 / 有色金属**：cycle + complex
- **养殖 / 农业**：cycle + medium

## 五、评分引擎（evaluate_stock）

### 周期股直接 pass
`CYCLE_STOCKS_ENABLED = False` —— `is_cycle` 的股票从 `get_month_signals` 返回结果中**完全过滤**，不出现在任何推荐列表。

### 非周期股 PE 判定（按行业区间）
- PE ≤ low → buy_heavy（重仓买入）
- PE ≤ (low+fair_low)/2 → buy_medium（中仓买入）
- PE ≤ fair_low → buy_light（轻仓买入）
- fair_low < PE ≤ fair_high → hold 或 buy_watch/sell_watch
- PE ≤ (fair_high+high)/2 → sell_light（关注卖出）
- PE ≤ high → sell_medium（适当卖出）
- PE > high → sell_heavy（大量卖出）

### ROE 门槛（按 complexity + 杠杆）
- simple: heavy=20, light=15, watch=12
- medium: heavy=22, light=17, watch=14
- complex: heavy=25, light=20, watch=15
- 杠杆调整：负债 <30% → -2pp，负债 >50% → +5pp（高杠杆行业豁免）
- ROE < watch → 信号降级为 hold
- ROE < light → 买入信号降级为 buy_watch
- ROE < heavy → buy_heavy/buy_medium 降为 buy_light

### 财务风险检查
- 负债率 > 70%（非高杠杆行业）→ 降级为 hold
- 毛利率 < 15%（非高杠杆行业）→ 降级为 hold

### 第一关：护城河趋势检查（check_moat）
触发任一即判松动，买入信号降级为 hold。

### 第二关：净资产收益率下降趋势检查
近 3 年 ROE 单调下降且总跌幅 ≥ 3pp 时，买入信号降级：
- 重仓 → 轻仓
- 中仓 → 轻仓
- 轻仓 → hold

防止"价值陷阱"（看起来便宜但趋势向下）。

## 六、护城河规则（check_moat_normal，8 条）

数据不足时返回 (True, []) —— 疑罪从无。

1. **最新亏损**：ROE < 0 → 松动
2. **ROE 单年暴跌**：连降 ≥ 6pp 且跌后 < 15%（巴菲特合格线）→ 松动
   - 跌后 ≥ 15% 豁免（五粮液 2013 从 36.8% 跌到 23.7% 被豁免）
3. **ROE 连续 3 年下滑**：且最新 < 15% → 松动
4. **毛利率连续 3 年下滑**：且累计跌幅 ≥ 5pp → 松动
5. **营收连续 2 年负增长**：且 ROE < 15%（双重证据）→ 松动
6. **负债率 + ROE 双重恶化**：负债率 3 年升 >10pp 至 >70% 且 ROE 连降至 <15% → 松动
7. **盈利质量恶化**：经营现金流/每股收益 连续 2 年 < 0.3 → 松动
   - 豁免 A：银行/保险/证券（含存款/保费波动）
   - 豁免 B：消费龙头（ROE ≥ 15% 且 毛利率 ≥ 50%）
8. **ROE 长期温水煮青蛙**：近 5 年单调下降 + 累计跌幅 ≥ 10pp + 最新 < 20%（巴菲特卓越线）→ 松动
   - 三个条件缺一不可，避免误杀"规模扩大后 ROE 自然正常化"的好公司
   - 触发案例：中国平安 2024 年（24→20→13→10→10%）
   - 豁免案例：贵州茅台 2017 年（45→39→32→26→24%，累计降 21pp 但最新 24%≥20%）

## 七、消费龙头现金流警示（get_cash_flow_warnings）

当规则 7 豁免触发时，输出多维度警示给持仓股：
1. 当前 ROE 和毛利率状态
2. ROE 单年跌幅
3. 毛利率单年跌幅
4. 营收同步下滑程度（行业周期 vs 单体造假）

实时选股模型还多 2 维：
5. 应收账款周转率（造假线索）
6. 存货周转率（造假线索）

警示状态变化才输出（首次触发/警示解除），避免日志嘈杂。

## 八、回测买卖逻辑（backtest_autorun.py）

### 买入
- 每月扫信号
- 对未持仓股票，按优先级排序：信号强度 > simple complexity > 高评分
- 按信号强度分配预算：
  - 重仓 30% × 可投资金
  - 中仓 15%
  - 轻仓 8%
- 兜底：比例算出不够 100 股但现金足够时，买 100 股（小资金友好）
- 最多同时持有 5 只（MAX_HOLDINGS）

### 卖出（严格按信号，不主观止损）
- **退市**：立即清仓
- **大量卖出**（sell_heavy，PE 远超行业上限）：立即清仓
- **适当卖出**（sell_medium，明显偏高）：
  - 非好公司：清仓
  - 好公司（历史 ROE 均值 ≥ 15%）：**豁免，继续持有**
- **护城河松动**：check_moat 触发任一规则 → 清仓

### 持有中不做的事
- 不加仓（买了就不动）
- 不主观止损（不因亏损 X% 就卖）
- 不因价格波动操作
- 不因市场情绪操作

### 分红
- 每月入账，按持股时长差别化扣税
- 分红金额直接加到 cash，可用于后续买入

## 十、实战验证（截至 2026-04-10）

10 个随机起始时间 × 4 档本金 = 40 次回测，全部正收益：
- 均值：1万 +165%、10万 +116%、50万 +109%、100万 +113%
- 最低 +8.8%（2019-10 起始，疫情+熊市）
- 最高 +267.8%（2012-01 起始，13.9 年）

好公司现金流测试（8 只消费龙头）：全部通过规则 7，无误杀。
万科案例：2021-2022 年经营现金流连续异常（比值 0.18、0.12），2022-06 起被护城河规则正确识别，避免后续巨亏。
五粮液案例：2014 年被规则 7 触发豁免，2015-04 输出"重点关注"警示，2016-04 警示解除。

## 十二、回购加分机制

巴菲特 2020 年股东信原话："苹果的投资**生动地说明了回购的力量**"。伯克希尔 2018 年买入苹果时持股 5.2%，到 2020 年苹果持续回购导致伯克希尔被动增持到 5.4%——一分钱没花的增持。

### 数据源
- `backtest_data/buybacks.json`：全股票池的回购历史（akshare `stock_repurchase_em` 一次性采集）
- 字段：start_date / status / amount / notice_date
- 只累计"实施进度=完成"的记录

### 评分规则（往前看 5 年）
- 累计回购金额 ≥ 50 亿 → +15 分（高加分，巴菲特最爱）
- ≥ 10 亿 → +8 分
- ≥ 1 亿 → +3 分
- 其他 → 0 分

### 在买入排序中的位置
**信号强度 > 简单生意 > 回购加分 > 高评分**

同等信号+同等简单度下，优先买有大额回购历史的公司。

### 典型评级（2024-06 时点）
- 🏅 格力电器（近5年 120 亿）、中国平安（110 亿）、伊利股份（68 亿）
- ⭐ 海尔智家（41 亿）、宝钢股份（36 亿）、中国石化（27 亿）
- （不加分）茅台、五粮液、招商银行、海天味业——近 5 年没有大额回购

### 效果
加入回购加分后，10 点稳定性测试的大本金均值提升显著（50 万和 100 万各提升约 28 pp）。

### 为什么不升级为硬门槛筛选（2026-04-10 决策记录）

**问题**：既然回购加分有效，为什么不把它从"排序维度"升级为"筛选硬门槛"（比如：没回购就不能重仓买入）？

**评估结论：不应该升级**。三层理由：

1. **时间维度的致命问题**：A 股回购文化在 2020 年后才兴起。
   - 2015-06 时点：70 只里只有 1 只有大额回购
   - 2020-06 时点：覆盖率仅 14%
   - 2025-06 时点：覆盖率 33%
   - 如果硬门槛存在，2015 年前的回测基本无法建仓

2. **股票维度的致命问题**：即使在回购文化最盛的 2024-06，70 只里仍有 **48 只没有大额回购**，这 48 只包括：
   - 贵州茅台（A 股第一白马）
   - 五粮液、招商银行、海天味业、片仔癀
   - 中国中免、恒瑞医药、云南白药
   这些是 A 股历史上收益最好的标的，硬门槛会把它们全部降级。

3. **巴菲特原话的严格解读**：巴菲特原文是"伟大公司的财务特征是长期稳定的高 ROE、高毛利率、**高分红或股票回购**"。关键字是"**或**"——二者任一即可。
   - 美股（苹果、微软）主要靠回购
   - A 股（茅台、五粮液）主要靠分红
   - **两种都算"回报股东"**，都符合巴菲特标准
   - 硬门槛等于强制"回购优于分红"，违背巴菲特原意

**最终决策**：保持现状（排序维度），回购作为加分项，不作为筛选门槛。已有的 +28pp 提升已经达到预期效果，进一步升级会带来更多风险而非收益。

**禁用方案**（未来 AI 也不要尝试）：
- ❌ 硬门槛（没回购就不能买重仓/中仓）
- ⚠️ 升级触发器（有回购则升级档位）需要先做对比测试才能决定

## 十三、未来可能的改进方向

1. **周期股二次确认机制**（已写但禁用）：连续 3 个月都判顶才卖
2. **盈利质量多年基准**：用近 10 年平均比值做基准，而非固定 0.3
3. **行业周期指标**：同行业对比，判断是公司问题还是行业问题
4. **非标审计意见识别**：从公告提取"非标准审计意见"作为造假线索
5. **关联交易占比**：过高的关联交易是造假预警
