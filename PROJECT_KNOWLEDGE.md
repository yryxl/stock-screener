## 一、项目目标

为个人价值投资者打造一个**完全被动**的选股通知系统：
- 用户不需要看大盘、不需要盯行情
- 系统每天自动分析，有买卖信号时微信通知
- 基于查理·芒格 + 沃伦·巴菲特的价值投资体系

## 二、核心投资理念（不可修改的原则）

### 2.1 选股三原则
1. **好生意**：简单易懂、轻资产、涨价只需换标签（如喜诗糖果）
2. **好公司**：高ROE、低负债、现金流充足、护城河宽
3. **好价格**：PE处于行业合理区间的低位

### 2.2 护城河判断（用户定义）
1. 用户离开该产品会**肉痛**（转换成本高）
2. 不会被同类产品的**性价比**打败
3. 产品依赖人性弱点或**习惯成瘾**

### 2.3 ROE标准（来自巴菲特1987年股东信）
- 10年均值 ≥ 20%，单年不低于15%
- 不依赖高杠杆（低负债+真现金流）
- ROE连续下滑破15% = 基本面恶化信号

### 2.4 生意复杂度分类
| 类型 | 特征 | ROE要求 | 举例 |
|------|------|---------|------|
| 简单生意 | 涨价换标签、低资本开支 | 宽松 | 白酒、中药、调味品 |
| 中等复杂 | 有技术门槛、中等资本开支 | 正常 | 电力、通信、医疗器械 |
| 复杂生意 | 重资产、技术变化快 | 严格 | 半导体、化工、矿业、装备制造 |

### 2.5 真假下跌判断
- **假跌**（买入机会）：基本面健康 + 同行业普跌 = 市场原因
- **真跌**（卖出信号）：利润下滑 + 现金流为负 + 只有它跌同行不跌

## 三、模型参数（当前版本）

### 3.1 ROE门槛（复杂度 × 杠杆）
```
基础门槛 = COMPLEXITY_ROE_ADJUST[complexity]
杠杆调整 = 负债率<30%时-2%, >50%时+5%
最终门槛 = 基础 + 杠杆调整

简单生意: 重仓≥20%, 轻仓≥15%, 关注≥12%
中等复杂: 重仓≥22%, 轻仓≥17%, 关注≥14%
复杂生意: 重仓≥25%, 轻仓≥20%, 关注≥15%
```

### 3.2 行业PE合理区间
```
银行: 6-9 | 白酒: 20-30 | 中药: 20-30
电力: 10-18 | 通信: 20-35 | 半导体: 40-65
化工: 12-20 | 矿业: 10-18 | 锂电: 30-50
（完整列表见 screener.py 的 INDUSTRY_PE 字典）
```

### 3.3 买入信号等级
```
PE ≤ 行业底部           → 可以重仓买入
PE ≤ (底部+合理下限)/2   → 可以中仓买入
PE ≤ 合理下限           → 可以轻仓买入
PE ≤ 合理区间中位偏低    → 重点关注买入
PE 在合理区间或以上      → 继续观望
```

### 3.4 卖出信号等级（仅持仓）
```
PE 合理区间             → 建议持续持有
PE 合理偏高             → 重点关注卖出
PE > 合理上限           → 可以适当卖出
PE 明显高于合理         → 可以中仓卖出
PE 远超行业上限         → 可以大量卖出
基本面恶化              → 基本面恶化警告
```

### 3.5 全市场筛选条件
- ROE 10年均值 ≥ 20%（剔除高杠杆）
- 负债率 < 50%，流动比率 > 1.5
- 营业利润率10年稳定或上升
- 自由现金流近5年为正
- 毛利率 > 40%（定价权）
- 股价 < 100元/股（1手≤1万元）

## 六、快照与回测

### 6.1 快照保存
- 每周自动保存一份到 `snapshots/` 目录
- 存储在GitHub仓库中 = 云端永久备份
- 包含：模型参数 + 推荐结果 + 股价 + 关注表 + 持仓

### 6.2 回测计划
- **3个月后**（2026年7月）：首次回测，验证模型初期表现
- **6个月后**：调整明显偏差的参数
- **1年后**：系统性评估，考虑是否需要大改
- **2-3年后**：长期验证芒格体系在A股的适用性
- **持续**：每半年回测一次

### 6.3 回测方法
1. 读取N个月前的快照
2. 查每只推荐股票当前价格
3. 计算：推荐时价格 → 当前价格的涨跌幅
4. 按信号等级分组统计命中率
5. 和沪深300同期对比（跑赢大盘才有效）

## 八、用户偏好记录

- 不看大盘，完全被动接收通知
- 1手预算≤1万元（股价<100元）
- 偏好简单生意（白酒、中药、消费品）
- 讨厌高杠杆公司
- 每日无论有无信号都发一条微信消息
- 关注表只显示买入方向信号，PE≥合理=观望
- 卖出信号只针对持仓
- 数据必须经过独立核查才能信任
- 朋友提供的分析需要先核查再采纳
